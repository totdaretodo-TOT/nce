# 新概念英语学习播放器 - 安装和运行指南

## 快速测试（无需安装 Node.js）

1. 直接在浏览器中打开 `test.html` 文件
2. 这是一个简化版本，可以立即使用

## 完整版本安装步骤

### 1. 安装 Node.js

访问 Node.js 官网下载并安装：
- 官网：https://nodejs.org/
- 推荐下载 LTS（长期支持）版本
- macOS 用户也可以使用 Homebrew 安装：
  ```bash
  brew install node
  ```

### 2. 安装项目依赖

在项目目录下运行：
```bash
npm install
```

### 3. 本地开发运行

```bash
npm run dev
```

然后在浏览器中打开显示的地址（通常是 http://localhost:5173）

### 4. 构建生产版本

```bash
npm run build
```

构建后的文件会在 `dist` 目录中

### 5. 预览生产版本

```bash
npm run preview
```

## 部署到 GitHub Pages

### 方法 1：自动部署（推荐）

1. 确保你的仓库设置中启用了 GitHub Pages
2. 进入仓库设置：Settings → Pages
3. Source 选择：GitHub Actions
4. 推送代码到 main 分支：
   ```bash
   git add .
   git commit -m "部署到 GitHub Pages"
   git push origin main
   ```
5. GitHub Actions 会自动构建和部署
6. 几分钟后访问：https://你的用户名.github.io/nce/

### 方法 2：手动部署

1. 本地构建：
   ```bash
   npm run build
   ```
2. 安装 gh-pages：
   ```bash
   npm install -g gh-pages
   ```
3. 部署：
   ```bash
   gh-pages -d dist
   ```

## 常见问题

### Q: 打开 index.html 显示空白？
A: 这是正常的，因为 Vite 项目需要通过开发服务器运行。请使用 `npm run dev` 或直接打开 `test.html`。

### Q: npm 命令找不到？
A: 需要先安装 Node.js，参考上面的安装步骤。

### Q: GitHub Pages 显示 404？
A: 检查仓库设置中的 Pages 配置，确保 Source 设置为 GitHub Actions。

### Q: 如何修改课程内容？
A: 编辑 `src/App.jsx` 文件中的 `BOOKS` 数组。
