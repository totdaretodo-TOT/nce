检测到地址栏是 file://，且你很可能打开的是项目根目录里的 index.html。

这个文件不能直接双击当网页用

根目录的 index.html 是给 Vite 开发/打包用的，浏览器会按失败路径去加载 /src/main.jsx，所以控制台会报 CORS / 找不到文件，页面是白的。

请改用下面任一方式：

  1. 本地开发预览：在终端进入本仓库目录，执行 npm run dev ，再用浏览器打开终端里显示的地址（例如 http://localhost:5173）。
  2. 离线打开（像「只双击网页」）：终端先执行 npm run build ，然后打开文件夹里的 dist/index.html （完整路径形如：…/nce/dist/index.html）。Mac 也可双击 open-dist.command。
