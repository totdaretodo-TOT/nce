# 📚 新概念英语学习播放器

一个现代化的新概念英语学习工具，支持逐句播放、词汇学习和课后测验。

## ✨ 特性

- 📖 **四册完整内容** - 涵盖 NCE 1-4 册课程
- 🎯 **逐句播放** - 支持单句和连续播放模式
- 📝 **词汇学习** - 每课配有重点词汇和例句
- ✅ **课后测验** - 即时反馈的互动测验
- 🌓 **深色模式** - 支持明暗主题切换
- 📱 **响应式设计** - 完美适配移动端和桌面端
- ⚡ **快速加载** - 基于 Vite 构建，性能优异

## 💻 本地开发

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/你的用户名/nce.git
cd nce

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

- **框架**: React 18
- **构建工具**: Vite 5
- **样式**: CSS Variables + 响应式设计
- **部署**: GitHub Pages + GitHub Actions

## 📝 使用说明

1. **选择课程** - 从顶部下拉菜单选择不同册别
2. **播放控制** - 使用播放/暂停按钮控制播放
3. **速度调节** - 点击速度按钮调整播放速度（0.5x - 2.0x）
4. **翻译模式** - 切换中英文显示模式（显示/隐藏/模糊）
5. **词汇学习** - 切换到"词汇"标签查看重点单词
6. **课后测验** - 切换到"测验"标签进行练习

## 🎨 主题

支持明暗两种主题，点击右上角的主题切换按钮即可切换。
